# ⚡ inazuma
ゲームパッドのボタンを自動連射するWindows用ツールです。

## 概要
仮想コントローラーを使用して、指定したボタンを一定間隔で連射します。
連射機能がないゲームでも、簡単に連射を実現できます。

## 機能
- ボタン選択（A, B, X, Y, LB, RB）
- 連射間隔の調整（100ms〜10000ms）
- ワンクリックで開始/停止
- 実行状態をタイトルバーに表示

## 必要環境
- Windows 10/11
- [ViGEmBus](https://github.com/nefarius/ViGEmBus/releases) のインストールが必要です

### ViGEmBus のインストール方法
1. [ViGEmBus Releases](https://github.com/nefarius/ViGEmBus/releases) にアクセス
2. 最新の `ViGEmBus_Setup_x64.msi` をダウンロード
3. インストーラーを実行
4. PCを再起動

## 使い方
1. `inazuma.exe` を起動
2. 連射したいボタンを選択
3. 連射間隔（ms）を設定
4. 「START」ボタンをクリック
5. 停止するときは「STOP」ボタンをクリック
